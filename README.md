# LevelUpLife Hi-Fi Prototype

This is the hi-fidelity prototype for the LevelUpLife mobile application, built for the INSY7311 Summative Project at The IIE.

LevelUpLife is a gamified productivity app aimed at students. Open `index.html` in any browser to view the screens.

 Built with Visual Studio Code using HTML and CSS and used an extension to see the outcomes of the screens
